# Herald

> Changelog and announcement generation from code evidence.

## Skills

| Command | What it does |
|---------|-------------|
| `/grounding-announcements` | Draft community announcements from actual shipping history — commits, PRs, releases. Every claim links to evidence. |
| `/synthesizing-voice` | Extract a voice profile from existing content so future communications stay consistent |
| `/chronicling-changes` | Produce structured change timelines from git history — what changed, when, and why |

## When to use

- You shipped a release and need to write the announcement
- You want to extract your team's writing style into a reusable voice profile
- You need a structured changelog from git history, not just `git log`
